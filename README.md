📊 FDA Drug Adverse Events Analysis Dashboard

📖 Project Overview
This project focuses on analyzing FDA drug adverse event data to identify patterns in drug safety, patient demographics, and serious outcomes.

The goal of this project is to perform end-to-end data analysis using Excel, SQL, and Power BI, and present insights through an interactive dashboard.

🎯 Objectives
- Analyze serious vs non-serious adverse events  
- Study age group distribution of patients  
- Understand gender-wise adverse event patterns  
- Identify trends over time  
- Highlight death-related cases  

## 🛠 Tools & Technologies Used
- Excel → Data Cleaning & Preprocessing  
- SQL → Data Analysis  
- Power BI → Data Visualization & Dashboard  

## 📊 Data Description
The dataset contains information about drug adverse events reported to FDA, including:

- Report Date  
- Serious (Yes/No)  
- Gender  
- Age  
- Drugs  
- Reactions  
## 🧹 Data Cleaning (Excel)
- Converted report_date into proper date format  
- Created new columns:
  - Serious Label  
  - Gender  
  - Age Group  
- Handled missing values  
- Created pivot tables for analysis  

## 🗄 SQL Analysis
- Created database and table  
- Imported Excel data into SQL  
- Used CASE statements for transformation  
- Performed analysis:
  - Serious vs Non-serious cases  
  - Age group distribution  
  - Gender vs Serious cases  
- Used GROUP BY and COUNT for aggregation
- 
## 📈 Power BI Dashboard
The dashboard includes:

- KPI Cards:
  - Total Cases  
- Visualizations:
  - Pie Chart → Serious vs Non-serious  
  - Bar Chart → Age Group Distribution  
  - Stacked Bar → Gender vs Serious  
  - Line Chart → Trend over time  
- Slicers:
  - Gender  
  - Age Group  
  - Serious Label  

## 🔍 Key Insights
- Higher number of serious cases observed  
- Certain age groups show more adverse events  
- Gender-based differences in reporting  
- Trends show variation in reporting over time
